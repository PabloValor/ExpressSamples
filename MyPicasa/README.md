# My Picasa

Sample inspired by https://www.youtube.com/watch?v=n3gtVjGYRLI

Simple search page using:

- Express
- Ejs
- Ejs-locals for layout
- Twitter Bootstrap
- https, url built-in package
- xml2js
- Picasa API

## Install

```
npm install
```

## Run

```
node app
```
It starts as `http://localhost:3000`.
